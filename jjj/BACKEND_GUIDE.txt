ğŸ“¡ API ENDPOINTS

Base URL: http://localhost:5000/api

1. Health Check
   GET /api/health
   Returns: Server status and version

2. Get Weather Data
   GET /api/weather?location=London
   Parameters: location (city name or coordinates)
   Returns: Complete weather and air quality data

3. Get Historical Data
   GET /api/weather/history?location=London&days=7
   Parameters: location, days (optional, default 7)
   Returns: Historical weather data

4. Search Locations
   GET /api/locations/search?q=London
   Parameters: q (search query)
   Returns: List of matching locations

5. Generate Alerts
   POST /api/alerts/generate
   Body: Weather data JSON
   Returns: Contextual alerts

6. Clear Cache
   POST /api/cache/clear
   Returns: Cache cleared confirmation

7. Get Statistics
   GET /api/stats
   Returns: Server statistics
